Linear Autoencoder¶

We first implement a linear autoencoder
. The LinearAE class construction receives two values:

  • d_in - the input dimension.
  • d_enc - the dimension of the encoding.

Each encoder saves two inner members:

  • self.encoder - an object of type nn.Linear implementing the encoding network.
  • self.decoder - an object of type nn.Linear implementing the decoding network.
In [2]:
class LinearAE(nn.Module):
    
    def __init__(self, d_in, d_enc):
        super().__init__()        
        self.encoder = nn.Linear(d_in, d_enc)
        self.decoder = nn.Linear(d_enc, d_in)
    
    def forward(self, x):
        return self.decoder(self.encoder(x))

Network Autoencoder¶

Next, we construct a NN-based autoencoder, using the following architecture:



All layers are linear, but note the dimensions in each layer, as well as the activation function (ReLU/Tanh).

The NetworkAE class construction receives four values:

  • d_in - the input dimension.
  • d_enc - the dimension of the encoding.
  • d_1 - the dimension of the first (and third) intermediate level.
  • d_2 - the dimension of the second intermediate level.

Each encoder saves two inner objects:

  • self.encoder - an object of type nn.Sequential implementing the encoding network.
  • self.decoder - an object of type nn.Sequential implementing the decoding network.
In [3]:
class NetworkAE(nn.Module):
    
    def __init__(self, d_in, d_enc, d1, d2):
        super().__init__()
        self.encoder = nn.Sequential(nn.Linear(d_in, d1),
                                     nn.Tanh(),
                                     nn.Linear(d1, d2),
                                     nn.ReLU(),
                                     nn.Linear(d2, d1),
                                     nn.ReLU(),
                                     nn.Linear(d1, d_enc))
        self.decoder = nn.Sequential(nn.Linear(d_enc, d1),
                                     nn.ReLU(),
                                     nn.Linear(d1, d2),
                                     nn.ReLU(),
                                     nn.Linear(d2, d1),
                                     nn.Tanh(),
                                     nn.Linear(d1, d_in))

    
    def forward(self, x):
        return self.decoder(self.encoder(x))

Training¶

Next, you implement a function that performs the training for a given network.

The input parameters are:

  • NN - the neural network (nn.Module) to train.
  • X - the training data (torch.Tensor).
    The dimensions of X are ($n\times d_{\mathrm{in}}$), where $n$ is the size of the training set, and $d_{\mathrm{in}}$ is the dimension of the network input.
  • learning_rate - rate of gradient descent steps.
  • max_iter - number of iterations to run.
  • print_iter - when to print status update.
In [4]:
def train_nn(NN, X, learning_rate, max_iter, print_iter=100):
    
    optimiser = torch.optim.SGD(NN.parameters(), lr = learning_rate)
    loss_fn = torch.nn.MSELoss(reduction='sum')
    
    for epoch in range(max_iter):
        
        X_pred = NN(X)
        loss = loss_fn(X_pred, X)
        
        if (epoch+1) % print_iter == 0:
            print(f'Epoch: {epoch+1}, Loss: {loss.item()}')
        
        optimiser.zero_grad()
        loss.backward()
        optimiser.step()
    
    return NN

For a comparison, we also implement a dimension reduction and reconstruction using principal component analysis.

from sklearn.preprocessing import StandardScaler

def pc_transform(X, PCS):
    X = X.T
    scaler = StandardScaler()
    scaler.fit(X.T)
    X = scaler.transform(X.T)
    U, _, _ = np.linalg.svd(X, full_matrices=False)
    result = np.dot((U[:,PCS]).T, X)
    return scaler.inverse_transform(result).T
